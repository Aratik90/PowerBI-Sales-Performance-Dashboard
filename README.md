# Sales Performance Dashboard - Power BI

## Project Overview
This project is an interactive Power BI dashboard designed to analyze sales performance across employees, products, categories, and regions.

## Dashboard Pages

### 1. Sales Overview
- Total Sales
- Total Profit
- Total Target
- Achievement %
- Monthly Sales Trend
- Regional Sales Analysis
- Category Contribution

### 2. Employee Performance
- Employee Sales Ranking
- Employee vs Target
- Profit Generated by Employee
- Employee Contribution %
- Performance Table

### 3. Product Analysis
- Top Products by Sales
- Profit by Product
- Product Contribution %
- Category Analysis
- Product Performance Table

## Tools Used
- Power BI Desktop
- DAX
- Power Query
- Excel

## Key KPIs
- Total Sales
- Total Profit
- Achievement %
- Employee Performance
- Product Performance
- Regional Analysis
